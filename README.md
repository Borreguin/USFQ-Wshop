### Resumen
- [Taller 1](/Taller1/README.md)
  - P1:
Problema del vendedor viajante (TSP): Consiste en encontrar la ruta más corta posible dada una lista de ubicaciones y sus distancias. El requisito es visitar cada lugar exactamente una vez y retornar al punto de origen, formando un ciclo cerrado.
  - P2:
El acertijo del granjero y el bote: Plantea el reto de un granjero que debe cruzar un río transportando un lobo, una cabra y una col. El bote solo tiene capacidad para el granjero (quien debe manejarlo) y una de sus compras a la vez. El objetivo es cruzar a todos sin riesgo, evitando dejar solos al lobo con la cabra o a la cabra con la col, ya que se comerían entre sí.
  - P3:
La torre de Hanoi: El objetivo es mover una pila completa de discos desde una torre de origen a una de destino. Para lograrlo, se deben seguir tres reglas estrictas:
-Solo mover un disco a la vez.
-Solo se puede tomar el disco superior de una pila para colocarlo en otra.
-Nunca se puede colocar un disco más grande encima de uno más pequeño.
